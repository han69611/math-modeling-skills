# 07-writing — 论文撰写（融合版）

本 skill 承接 `06-drawio`。目标是撰写一篇符合国赛一等奖标准的完整论文。

**本 skill 自动融合三套规范：**
- anti-ai-writing（15条降AI痕迹规则）
- cumcm-paper-writing（17条一等奖写作规范）
- visualization-review（12项图表质量审查）

## 必须产出

- `paper/main.typ` 或 `paper/main.tex`：完整论文源文件
- `paper/sections/*.typ` 或 `paper/sections/*.tex`：各章节文件
- 编译后的 `paper/main.pdf`
