# 图表质量审查（Visualization Review）

## 核心定位

图表的“质检员”。3coding-visual 负责画图，本 skill 负责检查所有图表是否达到国赛论文标准。

## 审查清单（12项检查）

1. 颜色统一性：同一含义的元素必须使用相同颜色
2. 字体一致性：全文字体大小、类型必须统一
3. 图例是否遮挡数据
4. 分辨率是否600dpi（推荐PDF矢量格式）
5. 网格是否过密
6. 标题是否写在图内（国赛由caption控制，不在图内写标题）
7. 中文是否乱码
8. 变量命名是否统一
9. Figure编号是否连续
10. 数据标签是否必要且清晰
11. 坐标轴是否规范（含单位、Y轴从0开始）
12. 图表与正文的对应

## 输出格式

逐项检查结果 + 问题明细 + 修复优先级
