# 04-coding-visual — 编程实现与图表生成

## 必须产出

- `code/utils.py`：通用工具函数
- `code/problem{N}.py`：各子问题代码
- `results/problem{N}.json`：各子问题结果
- `figures/*.pdf`：所有图表
- `reports/RESULTS_REPORT.md`：运行结果报告

## 工作流程

### Step 1: 搭建代码框架
### Step 2: 编写各子问题代码
### Step 3: 运行并验证
### Step 4: 生成论文级图表（含12项visualization-review检查）
### Step 5: 撰写结果报告

## 审批节点（必须执行）

1. 呈现关键数值结果和图表给用户
2. 标注与预期不符的结果
3. 等待用户确认后进入 `05-result-analysis`
